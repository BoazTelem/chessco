## Authoring discipline

- Keep §3 completion-based payout language consistent across Terms, Refund Policy, and any product copy.
- Banned vocabulary (spec §17): "battle / warrior / destroy / crush / hammer / smash / annihilate". Replace with neutral language.
- Reference specific spec sections in comments at the top of each file so legal review can cross-check.
- Drafts use placeholders like `{COMPANY_LEGAL_NAME}` and `{REGISTERED_ADDRESS}` — keep them obvious so the lawyer fills them in.
